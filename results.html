Results

This section should give a objective (unbiased) representation of trends, values, and calculations using charts, diagrams, graphs and other models to describe to the reader the data collected. In order to adequately do so, the results section needs to include these figures that are properly labeled with units for clear interpretation. Trends, values, and calculations should be visually shown through figures, but also reported in writing using explicit statements that capture the studentâ€™s understanding of the data presented.

Rubric

Construct Table, Graph, Chart (7 points)

  • Proper figure(s) are included to represent the data set. (1 point)
  • Title is provided and reflects the purpose of the data table. (1 point)
  • X-axis (Independent Variable) is labeled. (1 point)
  • X-axis has units. (1 point)
  • Y-axis (Dependent Variable) is labeled. (1 point)
  • Y-axis has units. (1 point)
  • A caption that briefly mentions or describes the control, independent variable, dependent variable and title of the table, graph and/or diagram. (1 point)

Report General Trends, Significant values, and Calculations (3 points)

  • Statement used to report the change in the dependent variable as the independent variable changes. (1 point)
  • Statement about significant values and outliers. (1 point)
  • Is there a column for a derived or calculated quantity that is named and has proper units? (Ex. average, density, volume, etc.)
  • Shows all calculations. (1 point)
    • Ex: F=m*a; d=m/v
